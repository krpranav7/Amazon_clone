This is a simple **Amazon homepage clone** built using **HTML and CSS**.

## 🚀 Live Demo
[Click here to view the project](https://krpranav7.github.io/Amazon_clone/)

## 📌 Features
- Amazon-style navbar
- Search bar
- Hero section
- Product category boxes
- Footer section
- Clean layout using HTML and CSS

## 🛠️ Tech Stack
- HTML5
- CSS3
- Font Awesome
